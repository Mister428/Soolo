# CamaraNoir

Un bot WhatsApp furtif.

## Déploiement

1. Clonez ce dépôt sur votre machine locale.
2. Installez les dépendances avec `npm install`.
3. Créez un compte sur Render.com.
4. Créez un nouveau Web Service sur Render et sélectionnez ce dépôt GitHub.
5. Configurez les variables d'environnement dans les paramètres du Web Service :
 - `PORT`: 3000
6. Déployez le service.

Le bot sera accessible après le déploiement.
